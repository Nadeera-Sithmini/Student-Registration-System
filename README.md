# Student Registration System

A web-based Student Registration System developed using Java, JSP, Servlets, and MySQL.

## Features
- Student Registration
- Student Login
- View Students
- Edit Student Details
- Delete Students
- Search Students

## Technologies Used
- Java
- JSP & Servlets
- MySQL
- Apache Tomcat
- HTML
- CSS
- JavaScript
- NetBeans IDE

## Project Structure
- src/ - Java source files
- web/ - JSP, HTML, CSS, JS files
- nbproject/ - NetBeans project files
